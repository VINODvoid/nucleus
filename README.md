## `> QUICK START_`

</div>

### Prerequisites

```bash
node --version  # v18+
bun --version   # v1.0+ (or npm)
```

### Installation

```bash
# Clone the nucleus
git clone https://github.com/VINODvoid/nucleus.git

# Enter the core
cd nucleus

# Install dependencies (choose one)
bun install
# or
npm install

# Launch the reactor
bun start
```
