## Image Guidelines

- Format: JPG or PNG
- Recommended size: 1080x2340 (mobile aspect ratio)
- Keep file sizes under 500KB for faster loading
- Use descriptive names for easy identification

## Adding Screenshots

Simply drag and drop your screenshot images into this folder and name them:
- `screen1.jpg` (or .png)
- `screen2.jpg` (or .png)
- `screen3.jpg` (or .png)

The main README.md will automatically reference these images.
